About the Site

This site is a collection of different graphs generated using d3.js, and a variety of custom data sources. Each graph will provide an example of the graph type generated in d3.js, as well as a description detailing the use cases and properties of the data that they visualize. Each data source that is displayed through the graphs was curated by hand for the purposes of having appropriate data sets for each graph type. This site serves to be an place to learn more about data visualization and the use cases of these charts.

Feel free to navigate through the site with the sidebar listing all of the different charts, and have fun learning! At the bottom of the navigation bar, there is a link to a page that implements a few graph types with some interactivity in order to do some proper data analysis.
