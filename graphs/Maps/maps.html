Map

Description

Usage Case

Maps display a location, and tend to have points to specify some specific data point regarding these locations. In this case, the circles on the graph are displaying important cities that I have visited. Add interactivity to display name of location on hover.

Marks

1 - Point

Channels

2 - Vertical and horizontal location, to denote where the specific city in question is.

Another potential channels could include color if you wish to differentiate between specific cities, and the size of each point to show another data point.
