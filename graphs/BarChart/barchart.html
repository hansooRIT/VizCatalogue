Bar Chart

Description

Usage Case

Bar charts tend to show comparisons between different categories of data. It is relatively easy to compare the sizes of different bars and notice the differences between them, so this is a popular means of displaying a comparison. In this case, the bar chart is used to show a change in calorie intake per day of the week for a person.

Marks

1 - Line

Channels

3 - Horizontal position, vertical position, color to denote if a data point is below a certain threshold (Calories < 2000)
