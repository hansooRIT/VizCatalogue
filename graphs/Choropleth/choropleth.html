Choropleth

Description

Usage Case

Choropleths are a type of map visualization where each different node has a different color to denote a different value. In this case, it is a scale to see what percentage of the U.S. population lives in the U.S., with darker states having higher populations. Need to add labeling.

Marks

1 - Area to denote the state/location in question

Channels

1 - Color, to display how much of the population is within each state. (Darker = larger proportion)
