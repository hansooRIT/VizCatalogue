Treemaps

Description

Usage Case

Treemaps visualize hierarchical data with a series of rectangles that are contained within each other. The root node has is the largest rectangle, and nodes with more children tend to be larger as well. Add labels and legend.

Marks

2 - Area to denote number of children, and containment to show the parent/child relationship between nodes. The area sizing within a treemap is often mapped from quantitative values and provides a part-to-whole relationship.

Channels

0 - No mandatory channels for this type of graph.

An optional channel would be color, if it is chosen to display a node's location in the hierarchy.
