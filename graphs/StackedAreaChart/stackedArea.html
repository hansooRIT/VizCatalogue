Stacked Area Chart

Description

Usage Case

Stacked area charts tend to describe a time-series relationship, but showing the proportion of different data values adding to a whole. In this example, each different color represents a different ice cream flavor sold, and the height of the top most portion of each line represents the total sales. There should be a legend denoting what each color represents as well. Add legend for color and interactivity.

Marks

1 - Area, to show how large the proportion of each data point is for that given time slot.

Channels

3 - Horizontal and vertical positions, and color to differentiate each flavor.
