Scatterplot

Description

Usage Case

Scatterplots use dots to represent data values derived from 2 different variables that are mapped to the x and y axes. They are usually used in order to see if there is a sort of correlation between the two data points, and are extremely useful in that regard. Clean up display a bit.

Marks

1 - Points

Channels

2 - Horizontal and vertical positions

Other potential channels could include color or area, as area could represent a third variable that is related to these, and color could differentiate between nodes if need be.
